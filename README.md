# 여기어땠어? (Frontend)

여기어땠어는 **여행 예산과 여행 후기를 기록·공유하는 게시판 서비스**입니다.  
이번 과제에서는 기존 “아무 말 대잔치” 게시판 프론트를 기반으로,  
여행 서비스 콘셉트에 맞게 **디자인과 UX를 커스텀**했습니다.

---

## 🎨 주요 변경 사항 (Frontend UI/UX)

- 서비스명 변경  
  - `아무 말 대잔치` → **`여기어땠어?`**
  - 헤더 타이틀: “여행 예산 & 후기 기록 보드”

- 라벤더 브랜드 컬러 적용  
  - CSS 변수(`:root`)로 라벤더 계열 색상 정의  
  - 버튼, 카드, 배경, 그림자까지 라벤더 톤으로 통일

- 로그인 화면 개편  
  - 좌측: 서비스 소개 텍스트  
  - 우측: 로그인 폼  
  - 사용자가 첫 화면에서 “어떤 서비스인지” 바로 인지할 수 있도록 구성

- 게시글 목록 화면  
  - 세로 리스트 → **카드 그리드 레이아웃**  
  - 각 게시글을 “여행 기록 카드”처럼 표현

- 게시글 상세 화면  
  - 상단: 좋아요 / 조회수 / 댓글 수 요약 박스  
  - 하단: 본문 + 댓글 영역  
  - 하나의 카드 레이아웃으로 자연스럽게 이어지도록 구성

- 공통 폼 레이아웃 통일  
  - 게시글 작성/수정, 프로필 수정, 비밀번호 변경 화면을  
    모두 중앙 카드형 레이아웃으로 통일

## ⚙️ 기술 스택

- **Vanilla JS (ES Modules)**
- **HTML / CSS**
- (백엔드 연동은 별도 REST API 기반, 추후 LLM 기능 연동 예정)

---

## 📁 주요 화면 / 파일 구조

- `html/index.html` : 로그인 페이지
- `html/board-list.html` : 게시글(여행 기록) 목록
- `html/board-detail.html` : 게시글 상세 조회
- `html/board-new.html` : 게시글 작성
- `html/board-edit.html` : 게시글 수정
- `html/profile.html` : 회원정보 수정
- `html/password.html` : 비밀번호 변경
- `css/style.css` : 전체 공통 스타일 (라벤더 테마)
- `js/pages/*.js` : 각 페이지별 화면 로직  
- `js/auth.js` : 인증 관련 공통 처리

(경로는 실제 프로젝트 구조에 맞게 조정해서 사용하면 됩니다.)
